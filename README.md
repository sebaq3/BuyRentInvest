# BuyRentInvest
plicación full‑stack (API + UI) para comparar escenarios inmobiliarios (comprar para vivir, alquilar e invertir, comprar para alquilar y vivir en otra vivienda más barata). Este documento ahora se centra en un backend modular y altamente testeable para que puedas ejecutar pruebas unitarias fácilmente.
